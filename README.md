# ios-homeworks

Учебный проект в рамках обучения по программе IOS-разработчик в Нетологии.

Приложение представляет из себя песочницу для выполнения практически всех домашних заданий на курсе. Не смотря на то, что цельного приложения не получилось в итоге, проект содержит много
полезной информации.
